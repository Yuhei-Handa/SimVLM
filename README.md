# SimVLM

SimVLM（Simple Visual Language Model）画像と自然言語から多言語画像翻訳、画像推論、画像認識など多様なタスクが可能なマルチモーダルAIです。

SimVLMのアーキテクチャを作成しました。下流タスクのヘッドは実装していないので、ご自由に追加することが出来ます。
